## Commands Used in GitBash and their Purpose
mkdir(make directory) = Creates a new Directory(Folder)
cp(copy) = copies file from one location to another.
rm(remove) = Delete a file permanently from the current directory.
ls(list) = Lists all files and folders in the current directory. Used to confirm creation, copying or deletion
cd(change directory) = Used to navigate between folders in the terminal.
touch(Touch) = Creates a new empty file.
vim(Vi IMproved) = Open or edit text files directly in the terminal.
